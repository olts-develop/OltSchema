Definition Type: Element
Name: OLTDossiers
Containing Schema: OLTDossiers.xsd
Abstract
Documentation: 
OLTDossiers is used to send Dossiers to the Agent Online and TO Online Web Services. At present only one "Dossier" can be sent in the request, but it is planned at a later stage, if needed, to allow the XML to contain more than one Dossier. Each dossier is mapped to one dossier in the back office.
