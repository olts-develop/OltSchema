Documentation for OLTDossiers
Definition Type: Element
Name: Dossiers
Containing Schema: Dossiers.xsd
Abstract
Documentation: 
This schema is used in more than one use case. In the use case "Display Dossier", a dossier in Agent Online is returned to the user. It is possible, that more than one Dossier is returned in the Dossiers node, Another use case is the import of an external dossier into an Agent Online to TO Online database. In this case the external developer sends one Dossier in the Dossiers node.
