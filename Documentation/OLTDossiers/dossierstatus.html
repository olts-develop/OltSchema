Abstract
Documentation: 
Status of the dossier. The textdata contains the full name of the dossier status. It does not need to be sent when importing a dossier, but a DossierStatusCode attribute code ought to be sent, so that the dossier will habe a corrent status. Example values for a Dossier Status are
OK
OFF - Offer
HOLD - On Hold
OP - Option
RQ - Request
XX - Cancellation
VORRES - Pre booking
If a dossier is sent from Agent Online, other states are possible:
OKINV - Dossier OK and ready for invoice printing
OKINVP - Dossier OK and has been printed
XXINVP - Dossier cancelled and the cancelled dossier has been printed
