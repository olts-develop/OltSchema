Abstract
Documentation: 
A dossier can be assigned codes that reference information provided by the customer:
PK - Employee ID
DS - Dept Code
KS- Cost Account
PR - Project no
IK - Account no
AU - Order no
A dossier can have more than one customer relevant reference code. As this information is specific to the customer Agent Online supplies 4 free text fields into which the relevant information can be written. The customer profile defines which fields in the dossier contain which data.
