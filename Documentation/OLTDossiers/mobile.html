Name: Mobile
Containing Schema: Customer.xsd
MinOccurs 0
MaxOccurs (1)
Abstract
Documentation: 
Mobile phone number of the customer. A complete telephne number can be sent as textdata, but it is also possible to send only the phone number in the textdata, and send the AreaCode separately as an attribute. It is advisable to send all phone or fax numbers as a complete string as the textdata or the relevant node and leave the AreaCode empty.
